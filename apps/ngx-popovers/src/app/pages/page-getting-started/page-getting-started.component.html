About

{{ '@ngx-popovers' }} is a library, implementing floating-ui.
With {{ '@ngx-popovers/core' }} you can create any dynamic component you need. But you can also use already ready-made component:
  • {{ '@ngx-popovers/tooltip' }}
  • {{ '@ngx-popovers/popover' }}

Install

Core package 

npm i {{ '@ngx-popovers/core' }}

Tooltip 

npm i {{ '@ngx-popovers/tooltip' }}

Popover 

npm i {{ '@ngx-popovers/popover' }}
